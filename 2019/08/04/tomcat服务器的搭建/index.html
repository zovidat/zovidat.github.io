tomcat服务器的搭建

2019-08-04

环境：

1、在网络上下载tomcat安装包（apache.fayea.com/tomcat）根据你的Java版本选择所对应的版本

2、在/usr/local内新建一个java文件夹，并下载之后将解压过的文件夹移动到其中

3、进入tomcat文件夹，再打开conf文件夹，vim server.xml 修改中的8080为80,保存退出

4、进入tomcat文件夹，再打开bin文件夹，输入./startup.sh启动tomcat服务

5、启动防火墙（systemctl start firewalld）将80端口添加到防火墙firewall-cmd –zone=public –add-port=8080/tcp –-permanent，重启防火墙，firewall-cmd –reload

6、打开浏览器，输入127.0.0.1进入tomcat服务器

7、进入conf文件夹打开用户配置文件。vim tomcat-user.xml，在倒数第二行输入以下代码，设置用户信息

8、进入bin文件夹，输入./shutdown.sh可关闭tomcat服务。
