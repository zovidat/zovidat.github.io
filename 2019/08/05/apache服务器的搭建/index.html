1、使用yum源安装apache（httpd）服务器

安装成功后会产生以下两个文件夹
/etc/httpd/ # httpd配置文件夹
/var/www/html/ # 网站的默认数据

2、打开配置文件，根据个人需求更改参数

3、启动apache服务器并将防火墙开启端口

查看端口的开放情况

4、设置apache服务器开机启动

5、在浏览器输入http://www.localhost.com:80 即可访问apache服务器的主页
