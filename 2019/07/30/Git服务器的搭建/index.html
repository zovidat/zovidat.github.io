git服务器的搭建

2019-07-30

环境：

一、在Linux上搭建git服务端系统。
1、安装git

2、Linux系统上创建 git 用户和用户组，用来管理 Git 服务，并为 git 用户设置密码

3、Linux系统上创建git仓库
在/home/git目录下下创建.ssh文件夹并修改文件夹的权限，之后在文件夹内创建authorized_keys文件并修改权限

4、在git用户家目录创建project目录，存放test.git项目

5、进入project目录，初始化git项目

二、客户端
在win10上安装git客户端 //git for Windows git for Linux
使用ssh创建私钥和公钥对

就会在家目录创建.ssh文件夹，里面会产生两个私钥和公钥文件
将公钥（.pub结尾的文件）内的内容写入到服务器的authorized_keys文件中
完成后测试克隆仓库到客户端机器
git clone git@ip:/home/git/project/test.git
